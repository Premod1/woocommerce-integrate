# WooCommerce ERP Integration Guide

This document outlines the connection between the Laravel ERP and the WooCommerce store using the official PHP SDK.

## 1. Prerequisites

To communicate with the WooCommerce REST API, you must install the official library via Composer:

```bash
composer require automattic/woocommerce
```
